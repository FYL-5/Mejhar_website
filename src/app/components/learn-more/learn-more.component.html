The increasing technological development has expanded the severity of cyber-attacks targeting institutions and individuals globally. Therefore, threat intelligence is considered a solid core element since it aims to provide regular and relevant data on cyber breaches, including local and global statistics. This project proposes the Mejhar tool, which gathers up-to-date information about potential cyber threats in the internet environment from dependable sources. It scans these warnings and then provides results to strengthen cybersecurity capabilities, which will be effectively used to mitigate cyber-attacks. The Mejhar tool's output is a website that gathers data about cyber threats from credible sources and then shows it via a dashboard with the severity of the attack, its source, name, and a ton of additional information with the option to search for keywords.
